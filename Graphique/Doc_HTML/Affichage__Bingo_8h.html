Documentation des macros

#define BINGO_VERIFIER_ALLOCATION ( pointeur,
message,
action   ) 

Valeur :

if (pointeur==NULL) {\
        affichage_erreur(message) ;\
        action\
}
Vérifie l'allocation d'un pointeur.

Paramètres:
pointeur Le pointeur à vérifier.
message Le message à afficher en cas d'erreur.
action L'action à effectuer après affichage du message d'erreur en cas d'erreur.
Le paramètre action doit être une ou plusieurs instructions C valides. Par exemple, BINGO_VERIFIER_ALLOCATION(ptr,"Erreur",return 1;) affichera "Erreur" et renverra 1.

Raffiche toutes les cases du bingo.

Paramètres:

Utilisé quand le joueur fait un MOTUS pour remettre l'affichage à 0. Pour mettre à jour la grille après que le joueur ait tiré un mot, préférer affichage_bingo_rafraichirLettres() qui est plus rapide.
